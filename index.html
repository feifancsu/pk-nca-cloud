📐 算法说明

AUC 计算方法

线性梯形法（默认）：AUC[i] = (C[i] + C[i+1]) / 2 × ΔT（全程适用）
线性升-对数降法：上升段用线性梯形；下降段（C[i+1] < C[i]）用对数梯形：
   AUC[i] = (C[i] − C[i+1]) / ln(C[i]/C[i+1]) × ΔT
AUC外推：AUC₀₋∞ = AUC₀₋t + C_last / λz

末端消除率常数 λz

末端对数-线性回归（ln C vs t），自动枚举所有可能点集（≥3点，均在Cmax之后）， 选择使调整R²最大的组合。t½ = ln(2) / λz

房室模型与拟合

使用RK4数值积分求解ODE系统，Levenberg-Marquardt算法最小化加权残差平方和：
WSSE = Σ wᵢ(C_obs,ᵢ − C_pred,ᵢ)²，wᵢ由选定权重方案决定
AIC = n·ln(WSSE/n) + 2p  |  BIC = n·ln(WSSE/n) + p·ln(n)
